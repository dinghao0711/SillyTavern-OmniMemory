# 🧠 SillyTavern-OmniMemory (全能导演与记忆引擎)

[![SillyTavern Extension](https://img.shields.io/badge/SillyTavern-Extension-blue.svg)](https://github.com/SillyTavern/SillyTavern)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

OmniMemory 是一个为硬核角色扮演（RP）设计的 SillyTavern 插件。它通过调用**独立的后台 API**，在不占用主线剧情上下文的情况下，静默提取你的状态、物品栏和关系网。

## ✨ 核心功能
- **📊 动态面板**：实时追踪天气、地点、时间。
- **🎒 物品与关系网**：自动记录你获得的物品（金币、武器）以及 NPC 对你的好感/关系变化。
- **🔄 自动模型拉取**：完美兼容 OpenAI, DeepSeek, 各种本地大模型。
- **🎬 剧情推荐器**：（开发中）卡壳时为你提供 3 个剧情走向。

## 📥 安装方法
1. 打开 SillyTavern，点击顶部的扩展（Extensions）图标。
2. 点击 **Install Extension**。
3. 将本仓库的 URL 粘贴进去并点击安装：
   `https://github.com/你的用户名/SillyTavern-OmniMemory`
4. 刷新页面即可在左/右侧边栏看到 OmniMemory 面板。
